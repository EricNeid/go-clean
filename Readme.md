# About

Simple tool to delete old files. Usefull for automatic cleanup via cronjob.

## Quickstart

1. Get go from <https://golang.org/dl/>

2. Download gobench

```bash
go get -u github.com/EricNeid/go-clean/cmd/goclean
```
